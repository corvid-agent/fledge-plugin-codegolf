# codegolf — fledge plugin

A fledge plugin

## Install

```bash
fledge plugins install ./codegolf
```

Or after publishing:

```bash
fledge plugins install owner/codegolf
```

## Commands

| Command | Description |
|---------|-------------|
| `fledge codegolf` | A fledge plugin |

## Development

Edit `plugin.toml` to configure commands, hooks, and capabilities.
See [fledge plugin docs](https://github.com/CorvidLabs/fledge) for the full plugin format.
